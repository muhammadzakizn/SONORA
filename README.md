# SONORA
Sonora adalah aplikasi web multimedia untuk relaksasi dan fokus melalui sound mixing interaktif. Tersedia di web dan Android, pengguna bisa membuat soundscape kustom dari suara alam (hujan, petir, angin, ombak) dengan intensitas yang dapat diatur. Fitur suara spasial 3D imersif dan kemampuan unggah musik pribadi juga tersedia.

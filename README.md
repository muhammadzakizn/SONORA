ğŸ’¡ Tentang Proyek

SONORA adalah sebuah prototipe aplikasi web multimedia yang berfokus pada relaksasi dan fokus melalui sound mixing (pencampuran suara) interaktif. Proyek ini dirancang sebagai alat bantu digital yang terpersonalisasi untuk membantu pengguna bersantai, meditasi, fokus kerja, atau tidur lebih nyenyak.

<br> Aplikasi ini menyelesaikan masalah umum seperti stres, gangguan kebisingan, dan kurangnya fokus dengan menyediakan soundscape (suasana suara) yang imersif dan dapat disesuaikan sepenuhnya oleh pengguna.

<br> Proyek ini dibuat sebagai bagian dari Mata Kuliah Proyek Aplikasi Berbasis Web di Politeknik Dewantara.
